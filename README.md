# autosetupvps
Just list of command to install LAMP on Linux Ubuntu

## Installation ##

1. Download the script
2. Apply permission to execute:

```
$ wget https://raw.githubusercontent.com/dianariyanto/autosetupvps/master/mantab.sh
$ chmod +x mantab.sh
$ sudo sh mantab.sh
```
